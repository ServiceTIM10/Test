# Pulizia file Excel rifiuti

Applicazione Streamlit per caricare un file Excel, pulirlo tramite lo script Python `pulisci_file_rifiuti.py` e scaricare il file elaborato.

## File principali

- `app.py`: interfaccia web Streamlit.
- `pulisci_file_rifiuti.py`: logica di pulizia Excel.
- `requirements.txt`: dipendenze Python per Streamlit Cloud.
- `.streamlit/config.toml`: configurazione base dell'app.

## Deploy su Streamlit Community Cloud

1. Caricare questi file in un repository GitHub.
2. Accedere a Streamlit Community Cloud.
3. Creare una nuova app collegata al repository.
4. Impostare `app.py` come main file path.
5. Cliccare Deploy.
